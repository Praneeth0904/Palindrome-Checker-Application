## 📌 Palindrome Checker

Checks whether a given word or phrase reads the same forwards and backwards.

---

## 🛠️ Tech Stack

HTML5 • CSS3 • JavaScript

---

## ✨ Features

* Checks whether a word or phrase is a palindrome.
* Ignores spaces and case sensitivity for accurate results.
* Provides instant validation feedback to the user.
